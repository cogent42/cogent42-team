A team brain that remembers across every teammate.

cogent42-team gives every teammate their own Telegram agent. Each agent talks to your team's Postgres knowledge base — projects, decisions, server facts, workflows — with per-fact ACLs, so you share what matters and keep the rest private. Gmail goes in automatically. You host the whole thing.

Get started View source

01How it works

Each teammate has their own bot container, provisioned by a control-plane on first onboarding. Bots write to and read from a single Postgres database. Gmail messages flow through the same extraction pipeline as chat.

INPUT Telegram + Gmail DM your bot. Sent emails ingest automatically.
→
EXTRACT Async worker Sonnet 4.6 distills facts; categorizes by ACL.
→
RECALL Postgres + pgvector Hybrid BM25 + cosine retrieval, injected into every prompt.

02What you get

The pieces a team install needs that a single-user assistant doesn't.

PER-FACT ACL

Share what matters, keep the rest private

Every fact is tagged private, team, or org. Personal preferences, rules, and sensitive content stay with the owner; project knowledge fans out.

GMAIL-AWARE

Sent mail flows in by default

OAuth-connected per user. Sent-only by default so you don't ingest cold inbound noise. PDFs, docx, xlsx, images get parsed too.

HYBRID RETRIEVAL

BM25 + pgvector cosine

Top-30 most relevant facts injected into the system prompt every turn. Own-knowledge bonus + permanence + evidence count rank ties.

SELF-HOSTED

One docker-compose, one host

Postgres + control-plane + extractor + gmail worker + per-user bot containers. No third-party SaaS to trust with your team's knowledge.

AUDIT

Every read and write logged

Cross-user fact reads, ACL flips, deletes — all in audit_log with actor, target, and reason. Replay anything.

CLAUDE AGENT SDK

Real tool use, not just chat

Each bot is the Claude Code preset with the team knowledge base as its memory. Reads files, runs commands, edits code — scoped per user.

03Get started

You'll need a host with Docker, a Postgres, an OpenAI key (embeddings), and a Telegram bot token per teammate. The setup script walks you through the rest. 

# clone
git clone https://github.com/cogent42/cogent42-team.git
cd cogent42-team

# interactive setup — generates .env, creates the first user
npm run setup

# bring everything up
docker compose up -d --build

# onboard a teammate (control-plane API)
curl -X POST -H "Authorization: Bearer $ADMIN_TOKEN" \
  http://127.0.0.1:8080/api/users \
  -d '{"email":"alice@org","telegram_bot_token":"…"}'

Full setup, architecture, and operations notes live in the README.
